### Tạo dự án
```bash
django-admin startproject main
```

### Tạo app
```bash
python manage.py startapp web
```

### Tạo migrations
```bash
python manage.py makemigrations
```

### Chạy migrate
```bash
python manage.py migrate
```

### Chạy server
```bash
python manage.py runserver 0.0.0.0:80
```

### Chạy test
```bash
pytest
```
